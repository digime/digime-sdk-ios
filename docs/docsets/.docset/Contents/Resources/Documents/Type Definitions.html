Type Definitions

The following type definitions are available globally.

  • DMEAuthorizationCompletion

    DMEAuthorizationCompletion - executed when authorization stage has completed.

    Declaration

    Objective-C

    typedef void (^DMEAuthorizationCompletion)(DMESession *_Nullable,
                                           NSError *_Nullable)

    Parameters

    session

    The session if authorization is successful, nil if not
    error

    nil if authorization is successful, otherwise an error specifying what went wrong
  • DMEOngoingAccessAuthorizationCompletion

    DMEOngoingAccessAuthorizationCompletion - executed when authorization stage for Ongoing Access has completed. Return to 3d party app.

    expiresOn - expiration date tokenType - string describing type: like Bearer

    Declaration

    Objective-C

    typedef void (^DMEOngoingAccessAuthorizationCompletion)(
    DMESession *_Nullable, DMEOAuthToken *_Nullable, NSError *_Nullable)

    Parameters

    session

    The session if authorization is successful, nil if not.
    oAuthToken

    OAuth data such as: access and refresh tokens, expiration date and type.
    error

    nil if authorization is successful, otherwise an error specifying what went wrong.

  • DMEFileContentCompletion

    DMEFileContentCompletion - executed when a file has been retrieved.

    Declaration

    Objective-C

    typedef void (^DMEFileContentCompletion)(DMEFile *_Nullable, NSError *_Nullable)

    Parameters

    file

    The file if retrieval is successful, nil if not
    error

    nil if retrieval is succesful, otherwise an error specifying what went wrong. The error’s user info will contain the id of the file this error relates to. e.g. NSString *fileId = error.userInfo[kFileIdKey];

  • DMESessionFileListCompletion

    DMESessionFileListCompletion - executed when getFileList response has changed.

    Declaration

    Objective-C

    typedef void (^DMESessionFileListCompletion)(DMEFileList *_Nonnull,
                                             NSArray<NSString *> *_Nonnull)

    Parameters

    fileList

    FileList object, representing latest file list snapshot.
    fileIds

    Array of string, fileIds. Only fileIds added, or updated since the last snapshot are included.
  • DMESessionDataCompletion

    DMESessionDataCompletion - executed when session data fetching has completed.

    Declaration

    Objective-C

    typedef void (^DMESessionDataCompletion)(DMEFileList *_Nullable,
                                         NSError *_Nullable)

    Parameters

    fileList

    FileList object, representing latest file list snapshot. This can be used to verify sync status state.
    error

    nil if session data fetching completed succesfully, otherwise an error specifying what went wrong.
